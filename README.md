# YouTube-Logo-Keychain

## Description
This project is a simple 3D YouTube logo keychain designed using Onshape.

## Features
- YouTube logo shape
- Key ring hole
- Rounded corners
- Ready for 3D printing

## Dimensions
- Width: 50 mm
- Height: 35 mm
- Thickness: 2 mm
- Hole Diameter: 4 mm

## File
- Logo.stl

## Software Used
- Onshape

## How to View
1. Download the STL file.
2. Open it using any 3D viewer.
